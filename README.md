# BRX Labz

A premium, high-performance web application designed for BRX Studios, built with modern web technologies to deliver an immersive user experience. This project showcases advanced animations, 3D elements, and a responsive design.

- **Framework**: [Next.js 15](https://nextjs.org/) (App Directory)

  - [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics**: [OGL](https://github.com/oframe/ogl)
- **Icons**: [Lucide React](https://lucide.dev/)

## üìÇ Project Structure

The project follows the standard Next.js App Router structure:

- `src/app`: Page routes and layouts.
- `src/components`: Reusable UI components.
- `src/sections`: Major sections of the landing page (Hero, Projects, Services, etc.).
- `src/data`: Static data files (FAQs, etc.).
- `public`: Static assets (images, fonts).
